presentations

The presentation should follow a storyline in order to have a natural flow. It is much easier for the public to follow a logical path as a top-down approach.

  • write a story with 1 slide per line
  • each slide must illustrate a single point
  • images are better, the public listens more if there's no text
    • illustrations should be done with vector drawing

It should be generated by LaTeX with the beamer package. A nice tutorial is available at http://heather.cs.ucdavis.edu/~matloff/beamer.html

A nice overview of what needs to be done for an academic talk: http://pages.cs.wisc.edu/~markhill/conference-talk.html
