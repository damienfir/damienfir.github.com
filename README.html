Static website generator

Dependencies

  • pandoc
  • sed

Usage

Script

Modify variables of the file generate.sh to suit your environment.

Syntax

The markup uses Markdown.

The links inter-documents have the syntax :file for the file you want to link to, and translates to [file](file.html) to suit the Markdown markup. For file in folder, the syntax is :folder/file.

Plain link such as [http://www.google.com](http://www.google.com) are translated to[http://www.google.com.](http://www.google.com](http://www.google.com).)
