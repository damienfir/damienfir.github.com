Classifying picture using EXIF

The idea is to use the EXIF data of a picture to categorize it amongst any possible class. The data is fetched on Flickr and clusters need to be found using any number of relevant EXIF parameters. An example application is to automatically assign tags to a picture.

Possible useful methods

K-means to find clusters (supervised), or ANN (unsupervised)

PCA to find the relevant parameters.
