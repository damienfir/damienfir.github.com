ctypes

ctypes mini tutorial

NumPy has an interface for ctypes compatibility with ndarray.
The ndpointer object needs to be initiated with a shape (tuple) to work, or the ndarray must be passed by reference with prior initialization (i.e. numpy.empty()). ndpointer should be used only for direct argtypes or restype, not in Structures and Unions.
Object passed by reference to a C function cannot have their address changed the usual way.

general tips

When coding with C and dynamic allocation, it is easier to avoid freeing the memory when debugging. That way there is less segfaults, and the code can be cleaned-up later.
When using incrementing variables, always initiate them to 0 just before the loop.
